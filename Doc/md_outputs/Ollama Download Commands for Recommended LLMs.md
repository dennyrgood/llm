## Page 1


**Ollama Download Commands for**

**Recommended LLMs**

These are the ollama run commands for each of the models recommended in the analysis, 

categorized by their primary use case. Running the command will download the model if it's 

not already on your system. 

**1\. Refactoring, New Code Creation, and Debugging**

Commands for models specialized in programming and complex reasoning tasks: 

ollama run qwen3-coder​

ollama run glm-4.6​

ollama run minimax-m2​

ollama run kimi-k2​

ollama run deepseek-r1​

ollama run firefunction-v2​

**2\. General Questions and Conversational Use**

Commands for robust models optimized for general chat, Q&A, and documentation: 

ollama run alfred​

ollama run notus​

ollama run qwen3​

ollama run goliath​

**3\. Manipulating and Understanding Images**

Commands for Vision-Language (VL) models: 

ollama run qwen3-vl​

ollama run gemma3​
